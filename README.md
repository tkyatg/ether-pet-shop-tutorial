# ether per shop tutorial

## what is this repository

- local でスマートコントラクトを試す tutorial。

## require

- npm
- solidity
- truffle

## 実行

- migration を行う

```
$ truffle develop

truffle(develop)>migrate
```

- meta mask で接続する
  `カスタムrpcでlocalの設定を行う`

- lite server 起動

```
npm run dev
```

- localhost:3000 で起動するので、metamask でサインインする
